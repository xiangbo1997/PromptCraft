# PromptCraft

一款 macOS 原生 AI 提示词优化工具，帮助你快速优化和管理 AI 提示词。

## 功能特性

- **智能优化** - 三种优化模式：简洁版、详细版、专业版
- **提示词库** - 保存、分类、标签管理你的提示词
- **历史记录** - 自动保存优化历史，方便回顾
- **收藏夹** - 收藏常用提示词，快速访问
- **菜单栏** - 常驻菜单栏，随时快速优化
- **快捷键** - 自定义全局快捷键
- **深色模式** - 支持系统主题切换
- **多语言** - 支持中文和英文

## 系统要求

- macOS 14.0 (Sonoma) 或更高版本

## 技术栈

- SwiftUI
- SwiftData
- Swift Observation
- KeyboardShortcuts

## 构建

```bash
xcodebuild -project PromptCraft.xcodeproj -scheme PromptCraft -configuration Release build
```
