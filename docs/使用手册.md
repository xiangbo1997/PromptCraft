## 简介

PromptCraft 是一款 macOS 原生应用，专为 AI 提示词优化设计。它可以帮助您：

- **优化提示词**：将您的想法转换为结构化、清晰的 AI 提示词
- **管理提示词**：保存、分类、收藏您的提示词模板
- **提高效率**：通过全局快捷键快速访问，支持菜单栏图标

### 系统要求

- macOS 14.0 (Sonoma) 或更高版本
- 需要有效的 OpenAI API Key（或兼容的 API 服务）

## 快速开始

### 1. 配置 API

首次使用前，您需要配置 AI 服务：

1. 打开应用，点击左侧边栏的 **设置** (Settings)
2. 在 **API 配置** 部分：
   - 输入您的 **API Key**
   - 如果使用非 OpenAI 服务，修改 **自定义 API 端点**
   - 等待 API Key 验证完成（显示绿色勾号）
3. 从下拉列表中选择您要使用的 **AI 模型**
4. 点击 **保存设置**

### 2. 优化您的第一个提示词

1. 点击左侧边栏的 **优化** (Optimize)
2. 在输入框中输入您的原始提示词或想法
3. 选择优化模式：
   - **简洁** - 精简到 50 字以内
   - **详细** - 完整保留所有信息
   - **专业** - 结构化专业格式
4. 点击 **优化** 按钮
5. 查看优化结果，可以：
   - **复制** - 复制到剪贴板
   - **保存** - 保存到提示词库
   - **重新优化** - 对结果再次优化

## 功能详解

### 提示词优化

#### 优化模式

| 模式 | 描述 | 适用场景 |
|------|------|----------|
| 简洁 | 将提示词精简到 50 字以内 | 简单问答、快速查询 |
| 详细 | 完整保留所有信息并优化结构 | 复杂任务、详细说明 |
| 专业 | 结构化格式，包含目标、背景、要求等 | 正式文档、专业写作 |

#### 操作说明

- **输入区域**：输入您的原始提示词，支持多行文本
- **输出区域**：显示 AI 优化后的结果（流式输出）
- **字符统计**：实时显示输入/输出的字符数及变化
- **交换按钮**：将输出内容移到输入区域，方便迭代优化

#### 统计信息

页面下方显示您的使用统计：
- 总优化次数
- 成功率
- 已保存的模板数量

### 提示词库

提示词库是您保存和管理所有提示词模板的地方。

#### 保存提示词

优化完成后，点击 **保存** 按钮即可保存到库中。

#### 管理提示词

- **搜索**：在搜索框中输入关键词快速查找
- **排序**：按名称、日期、使用次数排序
- **筛选**：按收藏状态、优化模式筛选

#### 批量操作

点击 **选择** 按钮进入批量模式：
- 多选提示词
- 批量删除
- 批量收藏/取消收藏

#### 导入导出

- **导出**：将所有提示词导出为 JSON 文件备份
- **导入**：从 JSON 文件导入提示词

#### 提示词卡片功能

每个提示词卡片支持：
- 点击查看详情
- 一键复制优化后的内容
- 添加/移除收藏
- 使用提示词（填充到优化页面）
- 删除

---

### 收藏夹

收藏您常用的提示词，便于快速访问。

#### 添加收藏

- 在提示词库中，点击卡片上的 ❤️ 按钮
- 或在提示词详情页点击收藏按钮

#### 搜索收藏

收藏夹支持独立的搜索功能，快速定位收藏的提示词。

---

### 历史记录

自动记录您的所有优化操作。

#### 时间分组

历史记录按时间自动分组：
- 今天
- 昨天
- 本周
- 本月
- 更早

#### 筛选功能

- **日期筛选**：选择特定日期范围
- **模式筛选**：按优化模式（简洁/详细/专业）筛选

#### 操作

- 查看历史记录详情
- 复制到剪贴板
- 从历史记录重新优化

---

### 标签与分类

使用标签和分类组织您的提示词。

#### 管理标签

1. 进入 **标签与分类** 页面
2. 点击 **添加标签** 创建新标签
3. 选择标签颜色
4. 为提示词添加标签

#### 管理分类

1. 点击 **添加分类** 创建新分类
2. 为分类添加描述
3. 将提示词归入分类

#### 按标签/分类筛选

在提示词库中，点击标签或分类名称即可筛选相关提示词。

### 设置

#### API 配置

| 设置项 | 说明 |
|--------|------|
| API Key | 您的 OpenAI API 密钥 |
| 自定义端点 | 如使用第三方服务，输入其 API 地址 |
| 模型选择 | 选择要使用的 AI 模型 |
| 超时时间 | API 请求超时时间（秒） |
| 重试次数 | 请求失败时的重试次数 |

#### 外观设置

| 设置项 | 说明 |
|--------|------|
| 主题 | 浅色/深色/跟随系统 |
| 语言 | 简体中文/繁体中文/English/日本語/跟随系统 |

#### 快捷键设置

自定义全局快捷键：
- **打开/关闭面板** - 默认 ⌘⇧P
- **快速优化** - 默认 ⌘⇧O
- **打开提示词库** - 默认 ⌘⇧L

#### 系统提示词

自定义每种优化模式的系统提示词，让优化结果更符合您的需求。

1. 点击 **编辑系统提示词**
2. 选择要编辑的模式
3. 修改提示词内容
4. 点击 **保存**
5. 可随时点击 **恢复默认** 重置

#### 其他设置

- **开机启动** - 登录时自动启动应用
- **复制提示** - 复制成功后是否显示提示

## 常见问题

### Q: API Key 验证失败怎么办？

**A**: 请检查：
1. API Key 是否正确复制（无多余空格）
2. 自定义端点地址是否正确
3. 网络连接是否正常
4. API 账户是否有余额

### Q: 优化结果不理想怎么办？

**A**: 您可以：
1. 尝试不同的优化模式
2. 在设置中自定义系统提示词
3. 使用"重新优化"功能迭代改进
4. 提供更详细的输入内容

### Q: 如何迁移数据到新电脑？

**A**:
1. 在旧电脑的提示词库中点击 **导出**
2. 将导出的 JSON 文件复制到新电脑
3. 在新电脑上点击 **导入**

### Q: 支持哪些 AI 服务？

**A**: PromptCraft 支持所有 OpenAI 兼容的 API 服务，包括：
- OpenAI 官方 API
- Azure OpenAI
- 其他兼容服务（如 DeepSeek、OpenRouter 等）

只需在设置中修改自定义端点即可。

### Q: 应用数据存储在哪里？

**A**: 数据存储在 macOS 的标准应用数据目录中：
- 提示词数据：使用 SwiftData 存储
- 设置：使用 UserDefaults 存储

### Q: 如何完全卸载？

**A**:
1. 将应用拖入废纸篓
2. 删除 `~/Library/Containers/com.yourcompany.PromptCraft`（如存在）
3. 清空废纸篓
